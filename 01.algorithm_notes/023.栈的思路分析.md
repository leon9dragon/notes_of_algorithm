## 栈的快速实现
### (1)使用数组模拟栈
用数组模拟栈的使用, 由于栈是一种有序列表,  
可以使用数组的结构来存储栈的数据内容.

****
### (2)实现思路分析
1. 使用数组模拟栈
2. 定义一个变量 top 表示栈顶指针, 初始值为 -1
3. 入栈的操作:  
   当有数据加入到栈的时候,  
   ```java
   //指针上移  
   top++;
   //给当前栈空间赋值
   stack[top] = data;
   ``` 

4. 出栈的操作:  
   当数据出栈的时候,  
   ```java
   //定义一个临时变量取的栈顶数据值
   int val = stack[top];
   //指针下移
   top--;
   //返回数据
   return val;
   ```
