## 从尾到头打印单链表
- 要求方式1: 反向遍历 
  - 解答: 反转链表如上一题, 但会破坏链表结构
- 要求方式2: Stack 栈

### 栈的思路
- 利用栈这个数据结构, 将各个节点压入到栈中  
  然后利用栈先进后出的特性, 实现逆序打印

### 栈的基本使用
- 如下所示的代码, 执行完毕后的输出结果是
```
String01
String02
String03
```
```java
public static void main(String args[]){
    //建立一个栈, 其存储的数据类型为String
    Stack<String> stack = new Stack();
    
    //向栈里面加入数据, 即入栈操作
    stack.add("String01");
    stack.add("String02");
    stack.add("String03");

    //取出栈里面的数据, 即出栈操作
    while(stack.size() > 0){
        //stack 的 pop 方法是将栈顶的数据取出的方法
        System.out.println(stack.pop());
    }
}
```
