# 稀疏数组 sparse array

## 1. 稀疏数组的实例
### 1.1 实际应用场景
需求:  
在五子棋程序中, 要能随时对棋盘进行存档和加载.

![五子棋](../99.images/2020-04-13-15-37-00.png)  

### 1.2 分析问题  
- 将五子棋棋盘转换成一个二维数组, 根据棋子颜色进行分类,   
  0 代表该点没有落子, 1 代表改点放的是黑子, 2 代表改点放的是蓝子.

- 最后得出了如上图右边的数组, 可以看到由于该二维数组很多纸默认是 0,  
  因此记录了很多没有意义的数据, 因而称这种数组为稀疏数组.
